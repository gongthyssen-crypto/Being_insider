# 真实历史走向对照

## 实际历史进程

- **1213年**：蒙古军攻入山东，李全家人遇害，聚众起义
- **1214年**：杨安儿称王，后败死；杨妙真率部与李全会合，结为夫妇
- **1218年**：李全率部归宋，任武翼大夫、京东副总管，部队改称"忠义军"
- **1218-1220年**：李全率军攻破莒州、密州、青州等地，活捉金将黄掴阿鲁答
- **1220年代**：李全势力继续膨胀，逐渐与南宋产生矛盾，对南宋朝廷的离心倾向加剧
- **1227年前后**：李全公开与宋敌对，转而依附蒙古
- **1231年**：李全在扬州兵败而死
- **后续**：杨妙真北渡淮水，与养子李璟归顺蒙古，名义上受蒙古招安，实际上保持独立

## 历史评价

- 李全从一个为家人复仇的起义者，逐步蜕变为在乱世中扩张个人势力的枭雄
- 红袄军的兴衰反映了金末山东地区复杂的地缘政治格局
- 李全的"两面通吃"策略（既依赖南宋名号又保持独立，后期又转而投蒙）最终导致多面树敌
- 杨妙真作为金末杰出的女军事领袖，所创梨花枪被戚继光推崇，"枪法之传，始于杨氏"
- 山东红袄军为日后金朝灭亡后蒙古在山东的统治提供了地方基础
